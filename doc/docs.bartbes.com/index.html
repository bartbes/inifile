love-misc-libs

Love-misc-libs is a repository of random libs I’ve created for LÖVE over the years. The following subprojects are also documented:

  • Grease: A networking library based on LuaSocket and Lua-ENet.
  • Async: A library for asynchronous calls in LÖVE.
  • Inifile: An ini file parser and writer.

Grease and inifile also work without LÖVE.
